# Titanic Data Cleaning & Analysis

## Objective
The goal of this project is to clean and analyze the Titanic dataset.

## Dataset
Titanic dataset from Kaggle.

## Steps Performed
- Data cleaning
- Handling missing values
- Removing unnecessary columns
- Converting categorical data into numerical format
- Data visualization

## Key Insights
- Females had a higher survival rate than males
- Most passengers were between age 20–40
- Majority of passengers did not survive

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
